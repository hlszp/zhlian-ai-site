# Content model and editorial intent

This directory is reserved for the next content migration: moving cases, principles,
papers, repositories, and vendor entries out of React components into validated
Markdown or structured data files.

Until that migration is complete, production content remains in `src/App.tsx` and
styles remain in `src/styles.css`. New entries should have a clear category, a
short evidence-based description, a source link when applicable, and a review date
when the information can change.

The editorial goal is a compact ZHILIAN knowledge reference for process-industry AI
and agents. Keep the tone factual and practical. Avoid unsupported rankings,
customer claims, political or corporate slogans, and promotional organization names
until the project explicitly adopts them.
